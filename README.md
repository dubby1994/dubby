## eureka

本地测试时：

修改hosts文件：

```
127.0.0.1       peer1
127.0.0.1       peer2
```

```
java -jar eureka-0.0.1-SNAPSHOT.jar --spring.profiles.active=peer1
```

```
java -jar eureka-0.0.1-SNAPSHOT.jar --spring.profiles.active=peer2
```

访问[http://peer1:8761/](http://peer1:8761/)，[http://peer2:8762/](http://peer2:8762/)
